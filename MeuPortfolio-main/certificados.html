Cursos Concluídos

Cursando Engenharia de Software

Aprendi sobre fundamentos e arquitetura de computadores, redes de computadores, engenharia de software, matemática aplicada a computação, algoritmos e lógica dentre outras matérias.

JavaScript Completo

Esse curso vai do básico ao avançado de JavaScript e ao final dele eu tive uma compreensão total da linguagem, dos seus principais métodos e da sua sintaxe. O foco é no aprendizado da linguagem, mas o principal local de aplicação foi na web, através de manipulações do DOM e requisições via Fetch API.

HTML e CSS para iniciantes

Neste curso aprendi do 0 sobre HTML e CSS que é a base para o desenvolvimento Front-End, nele pude aprender tags semânticas, seletores de CSS, posicionamento, responsividade e com ele construi um projeto real chamado Bikcraft.

Wordpress REST API Dogs

Neste curso eu aprendi a utilizar a REST Api do WordPress. O objetivo do curso foi a criação de endpoints customizados e específicos para o projeto do curso de React Completo. Apenas um plugin foi utilizado no curso, para permitir a autenticação de usuários utilizando JSON Web Tokens.

Bootstrap 4

Este curso aborda o Bootstrap na versão 4.0, porém isso me ajudou a ter que estudar a documentação e consequentemente aprender mais a fundo esse framework na sua versão mais recente que pode deixar o desenvolvimento de sites muito mais ágil. E com a conclusão do curso acabei também o site Caravan, totalmente funcional.

CSS FlexBox

Neste curso aprendi melhor a criar layouts responsivos utilizando apenas propriedades do Flexbox. No final dele conclui o projeto FlexBlog.
